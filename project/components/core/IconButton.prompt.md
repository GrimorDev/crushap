Circular icon-only action button — used for swipe actions (pass/like/superlike), nav, and toolbar controls.
```jsx
<IconButton icon="heart" variant="filled" label="Like" />
<IconButton icon="x" variant="outline" label="Pass" />
```
Always pass a `label` for accessibility. Sizes: sm (36), md (44), lg (56, used for primary swipe actions).
